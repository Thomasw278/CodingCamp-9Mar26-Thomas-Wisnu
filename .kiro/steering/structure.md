# Project Structure

## Directory Organization

```
.
├── .git/           # Git version control
├── .kiro/          # Kiro AI assistant configuration
│   └── steering/   # AI guidance documents
├── .vscode/        # VSCode workspace settings
└── README.md       # Project documentation
```

## Conventions

The project follows a standard repository structure with:
- Configuration files at the root level
- Kiro steering rules in `.kiro/steering/` for AI assistant guidance
- VSCode settings in `.vscode/` for editor configuration

As the project grows, additional directories for source code, tests, and documentation should be added following standard conventions for the chosen technology stack.
